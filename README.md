# targzme
Too many files? Compress specific subfolders recursively in a breeze!
Recursively finds sub-folders, compresses them, and optionally deletes originals.


## Installing

Copy `targzme` to your `~/bin/` folder, create it if it doesn't exist already:
```shell
mkdir ~/bin
cp targzme ~/bin
```

Check that it works by printing the help message:
```shell
targzme -h
```
